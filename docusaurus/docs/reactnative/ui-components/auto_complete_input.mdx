`AutoCompleteInput` is a UI component to render the input box. This component provides autocomplete functionality for the commands and mentions feature.
Commands suggestions will show up when user types "`/`", while mentions suggestions will show up when user types "`@`".

Please check the guide of [Customize Message Input](../guides/message_input.mdx) for usage of this component.
