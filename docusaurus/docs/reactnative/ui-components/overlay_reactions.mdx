`OverlayReactions` is a UI component to render list of existing reactions within message overlay, which opens up when user long presses a message.

You can replace this default UI component using the `OverlayReactions` prop on [Channel](../core-components/channel.mdx) component.

```tsx
<Channel OverlayReactions={CustomOverlayReactions} />
```

## Props

### <div class="label required">required</div> **alignment** 

<Alignment />

### reactions

List of existing reactions, which are extracted in following format:

```tsx
const reactions = message.latest_reactions.map((reaction) => ({
  alignment:
    clientId && clientId === reaction.user?.id ? 'right' : 'left',
  image: reaction?.user?.image,
  name: reaction?.user?.name || reaction.user_id || '',
  type: reaction.type,
}))
```
