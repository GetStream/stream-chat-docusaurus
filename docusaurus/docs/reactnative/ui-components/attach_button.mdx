`AttachButton` is a default UI component for rendering attachment button within `MessageInput` component. When user presses this button, it opens the attachment picker bottom sheet.

You can set `AttachButton` prop on Channel component to replace this default component:

```tsx
<Channel AttachButton={CustomAttachButton} />
```
