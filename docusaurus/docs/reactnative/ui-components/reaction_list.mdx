`ReactionList` is a UI component to render reactions added on message, on top of message. This is a child component of `MessageSimple`.

## Basic Usage

**Use Case** Override background color for reaction list container:

```tsx
import { Channel, ReactionList } from 'stream-chat-react-native';

const ReactionListWithCustomBackground = () => <ReactionList fill={'#00DDD'} stroke={'#fffff'} />

<Channel ReactionList={ReactionListWithCustomBackground} />
```
