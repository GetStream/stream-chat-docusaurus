`MoreOptionsButton` is a default UI component for a button, which when pressed toggles the visibility of `InputButtons`.

You can replace this default UI component using the `MoreOptionsButton` prop on [Channel](../core-components/channel.mdx) component.

```tsx
<Channel MoreOptionsButton={CustomMoreOptionsButton} />
```

## Props

### disabled

_(Overrides the value from [ChannelContext](../contexts/channel_context.mdx))_
