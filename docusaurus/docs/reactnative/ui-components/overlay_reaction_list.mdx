UI component to render the reaction selector within message overlay.

You can replace this default UI component using the `OverlayReactionList` prop on [Channel](../core-components/channel.mdx) component.

```tsx
<Channel OverlayReactionList={CustomOverlayReactionList} />
```

### data

_(Overrides the value from [MessageOverlayContext](../contexts/message_overlay_context.mdx#data))_
