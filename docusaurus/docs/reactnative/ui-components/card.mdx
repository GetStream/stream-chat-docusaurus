`Card` is a UI component used by library to render custom attachment types.

Please check the guides for [Custom Attachment](../guides/attachment_customizations.mdx) for details.

## Props

### additionalTouchableProps

_(Overrides the value from [MessagesContext](../contexts/messages_context.mdx))_

<AdditionalTouchableProps />

### author_name

Author name, to be rendered in footer of card.

| Type |
| - |
| string |

### image_url

Url of image to render as cover image for card. In absence of `image_url`, `thumb_url` will be used for cover.

| Type |
| - |
| string |

### og_scrape_url

URL to redirect to, when user presses on card.

| Type |
| - |
| string |
