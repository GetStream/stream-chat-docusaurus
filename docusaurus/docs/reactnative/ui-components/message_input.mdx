`MessageInput` component provides all the UI and necessary functionality (e.g, attachment picker, commands, mentions autocomplete etc.) for composing a message.
This component must be used as child of `Channel` component.

## Basic Usage

```tsx
<OverlayProvider>
  <Chat client={client}>
    <Channel channel={channel}>
      <MessageInput />
    </Channel>
  </Chat>
</OverlayProvider>
```

:::note
All the configuration for `MessageInput` can be done on [`Channel`](../core-components/channel.mdx) component. Additionally please take a look at [Guide Section](../guides/message_input.mdx) about customizing MessageInput UI
:::
