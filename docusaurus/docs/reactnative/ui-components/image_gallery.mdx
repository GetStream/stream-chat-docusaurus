`ImageGallery` is a UI component for image viewer/gallery feature. This component handler all the gestures for zooming and swiping on images within gallery.

Its not possible to replace this default UI component, given the complexity of it. But you can use `imageGalleryCustomComponents` prop on [`OverlayProvider`](../core-components/overlay_provider.mdx) to replace sub-parts of image gallery.
