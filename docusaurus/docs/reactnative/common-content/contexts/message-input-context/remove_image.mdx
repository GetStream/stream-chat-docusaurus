Removes attached image in message, which is being composed in `MessageInput` component. By default, this function is attached to onPress handler for close button in [ImageUploadPreview](/reactnative/core-components/channel#imageuploadpreview).
It takes index of file (to be removed) in [imageUploads](/reactnative/contexts/message-input-context#imageuploads) array as parameter.

| Type |
| - |
| `(index) => void` |
