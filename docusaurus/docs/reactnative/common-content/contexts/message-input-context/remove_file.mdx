Removes attached file in message, which is being composed in `MessageInput` component. By default, this function is attached to onPress handler for close button in [FileUploadPreview](/reactnative/core-components/channel#fileuploadpreview).
It takes index of file (to be removed) in [`fileUploads`](/reactnative/contexts/message-input-context#fileuploads) array as parameter.

| Type |
| - |
| `(index) => void` |
