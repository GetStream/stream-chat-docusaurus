## Customization

The message list uses the `Message` component to display messages, if you want to replace that with your own custom component, here is how to do it:

**Example 1** - custom message component:

```html
<stream-channel>
  <stream-channel-header></stream-channel-header>
  <stream-message-list
    [messageTemplate]="customMessageTemplate"
  ></stream-message-list>
  <stream-message-input></stream-message-input>
  <stream-thread name="thread">
    <stream-message-list
      name="thread-message-list"
      mode="thread"
      [messageTemplate]="customMessageTemplate"
    ></stream-message-list>
    <stream-message-input
      mode="thread"
      name="thread-message-input"
    ></stream-message-input>
  </stream-thread>
</stream-channel>

<ng-template
  #customMessageTemplate
  let-message="message"
  let-enabledMessageActions="enabledMessageActions"
  let-canReactToMessage="canReactToMessage"
>
  <app-custom-message
    [message]="message"
    [enabledMessageActions]="enabledMessageActions"
    [canReactToMessage]="canReactToMessage"
  ></app-custom-message>
</ng-template>
```

Your custom template can receive the same inputs (with the same name) as the default `Message` component.

:::note

If you want to build your own `Message` component, you might find the following building blocks useful:

### Avatar

The `Message` component uses the [`Avatar`](./AvatarComponent.mdx) component to display the sender's photo.

### Message actions

The `Message` component uses the [`MessageActionsBox`](./MessageActionsBoxComponent.mdx) component to enable interaction with the message (i.e. edit).

### Message reactions

The `Message` component uses the [`MessageReactions`](./MessageReactionsComponent.mdx) component to display reactions of a message, and enable reacting to a message.

### Attachments

The `Message` component uses the [`AttachmentList`](./AttachmentListComponent.mdx) component to display the attachments of the message.

### User firendly dates

The `Message` component uses the [`parseDate`](https://github.com/GetStream/stream-chat-angular/tree/master/projects/stream-chat-angular/src/lib/message/parse-date.ts) utility function to create user-firendly dates (for example _Yesterday at 8:15 AM_).

### Read by

The `Message` component uses the [`getReadByText`](https://github.com/GetStream/stream-chat-angular/tree/master/projects/stream-chat-angular/src/lib/message/read-by-text.ts) utility function to display the list of people who have read a particular message in a user friendly way (for example _Bob, Sophie, Jack, Rose, John and 1 more_).

### Loading indicator

The `Message` component uses the [`LoadingIndicator`](./LoadingIndicatorComponent.mdx) component to display a spinner while the message is being sent.

### Icons

The `Message` component uses the [`Icon`](./IconComponent.mdx) component to display icons.
:::
